gstl
=======
利用C++11实现的一个简易版的stl

目的：掌握stl，练习数据结构与算法和C++ Template编程

编译环境：GCC4.0 以上

### 开发计划：
  * STL的几大基本组件，如string、vector、list、deque、set、map、unordered_\*等
  * STL算法库的大部分算法
